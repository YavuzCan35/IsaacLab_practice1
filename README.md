# IsaacLab_practice1
Isaaclab is a robotics learning tool by NVIDIA. Focused on mainly reinforcement learning and paralellization of training, athereby accelerating the training thousands times.


1- Create 3d Cad drawing in fusion360.

2- Merge components into base_link,other_link1, other_link2 etc.

3- Use urdf export script from github for fusion360.

4- Change the file ending with .xacro to .urdf

5- Open isaacsim and import the urdf(select movable base if robot is mobile)

6- Add contact reporter from link prim by right click and save.

7- If rough terrains gonna be used then height scanner is required.
